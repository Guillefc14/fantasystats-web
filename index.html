Política de Privacidad

Última actualización: abril 2026

1. Información que recopilamos

FantasyStats no almacena ni recopila datos personales en nuestros servidores. La aplicación actúa como intermediario entre el usuario y la API oficial de La Liga Fantasy.

Las credenciales de acceso se transmiten directamente a los servidores de La Liga Fantasy para autenticación y nunca se almacenan en nuestros servidores. Los tokens de sesión se guardan de forma segura en el dispositivo del usuario.

2. Uso de la información

La información obtenida de La Liga Fantasy se utiliza exclusivamente para mostrar los datos de tu equipo, generar recomendaciones de fichajes y calcular el 11 recomendado.

3. Compartición de datos

No compartimos ningún dato personal con terceros. No vendemos datos de usuarios.

4. Seguridad

Los tokens de autenticación se almacenan en el almacenamiento seguro del dispositivo (Expo SecureStore) y se transmiten siempre mediante HTTPS cifrado.

5. Servicios de terceros

  • La Liga Fantasy API (api-fantasy.llt-services.com) — datos del juego
  • Azure Active Directory B2C (login.laliga.es) — autenticación

6. Contacto

Para cualquier consulta sobre privacidad: ffantasystats@gmail.com

7. Cambios en esta política

Notificaremos cualquier cambio significativo mediante una actualización de la aplicación.
